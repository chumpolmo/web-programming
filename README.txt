สำหรับโปรแกรมตัวอย่างเป็นการนำเสนอรูปแบบของการดำเนินการบนฐานข้อมูล MySQL ร่วมกับภาษา PHP เบื้องต้นเท่านั้น
โดยก่อนทดสอบการทำงาน ผู้พัฒนาควรดำเนินการ ดังนี้
1. สร้างฐานข้อมูลและนำเข้าข้อมูลในฐานข้อมูล MySQL โดยนำเข้าจากเอกสาร customer.sql ภายใต้โฟลเดอร์ database
2. กำหนดข้อมูลการเชื่อมต่อฐานข้อมูลในเอกสาร DBController.php ภายใต้โฟลเดอร์ controller
3. Please enjoy.
