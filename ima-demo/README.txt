---------------------------------------------------------------------------------------------------------------------------
-- คุณสมบัติเบื้องต้นของเว็บแอปพลิเคชัน --
- การเข้าสู่ระบบ/ออกจากระบบ โดยตรวจสอบการเข้าสู่ระบบ/ออกจากระบบ ด้วยบัญชีผู้ใช้ในฐานข้อมูล
- การจัดการข้อมูลข่าว (เพิ่ม/ลบ/แก้ไข) เฉพาะบัญชีผู้ใช้ในฐานข้อมูล
- การจัดการข้อมูลผู้ใช้ (เพิ่ม/ลบ/แก้ไข) เฉพาะบัญชีผู้ใช้ในฐานข้อมูล
- การแสดงผลข้อมูลอุปกรณ์ไอโอที (หรืออื่น ๆ) และอัปเดตรูปภาพอุปกรณ์ โดยควบคุมการใช้งานด้วยอาเรย์ (Array)
- การควบคุมการแสดงผลเมนู ด้วยสิทธิ์ของผู้ใช้ที่เข้าสู่ระบบ/ผู้ใช้ทั่วไป
- การควบคุมการดำเนินการของรายการข่าว (เพิ่ม/ลบ/แก้ไข) ด้วยสิทธิ์ของผู้ใช้ที่เข้าสู่ระบบ/ผู้ใช้ทั่วไป

-- การติดตั้งซอฟต์แวร์ --
1. คัดลอกไฟล์ไปยังตำแหน่งเอกสารบนเว็บเซิร์ฟเวอร์ (ในกรณีที่ติดติ้งบนเครื่องเซิร์ฟเวอร์ที่มีให้ Apache Web Server ให้บริการ)
2. สร้างฐานข้อมูลและนำเข้าฐานข้อมูลด้วยไฟล์ database/ima-demo.sql และเพิ่มเติมข้อมูลในระบบฐานข้อมูลได้ตามความเหมาะสม
3. กำหนดค่าการเชื่อมต่อฐานข้อมูลในไฟล์ libs/function.php สำหรับการเชื่อมต่อระบบฐานข้อมูล
3. กำหนดค่า config หรือเพิ่มเติมข้อมูลในไฟล์ libs/config.php เพื่อการแสดงเนื้อหาบนเว็บที่สมบูรณ์ สามารถเพิ่มเติมข้อมูลได้ตามความเหมาะสม
4. ทดสอบการใช้งานเว็บผ่านทาง http://localhost/ima-demo (ในกรณีใช้งานบนเครื่องคอมพิวเตอร์ด้วยเซิร์ฟเวอร์จำลอง)
